第一天

Kwal、Pok 和 Mux 是三个彪形大汉。 他们能打善斗，身高都在两米以上，非常吓人。 
远处，他们看到一座孤零零的农舍坐落在平原上。 
他们前往那里，发现了一个家庭：父亲、母亲、两个儿子、一个女儿和两个祖母。 Kwal、Pok 和 Mux 展示了他们的肌肉和武器，然后要求一顿热饭。 母亲和女儿照做了，而父亲和两个儿子则感到尴尬。

Kwal、Pok 和 Mux 对家人说： “要么我们把你们全杀了， 要么你们每天给我们准备三顿丰盛的饭菜！” 家人毫不犹豫地同意了，每天为他们准备三顿饭。 Kwal、Pok 和 Mux 回家了。

家人思考如何避免失去这么多辛苦生产的食物， 只为养活这三个怪物。 母亲提议毒死他们，她准备了毒药，等待这三头怪物。

Kwal、Pok 和 Mux 回来了，但这次他们检查了整个农场， 并命令母亲和女儿跟他们回家做饭， 而家里的其他人则要为他们生产食物。 父亲和儿子试图反抗，但力量悬殊， 他们只好勉强同意。

三个壮汉的家人接待了农妇： 女人们照顾她们，给她们安排了房间 还有浴室让她们洗澡和换衣服。 她们将听从 Kwal 的妻子的指挥，她是一位出色的厨师。 在这里，根本无法下毒。

第二天

Kwal、Pok 和 Mux 在地区里搜寻农场。 每找到一个，他们就用武力威胁并索要食物。 
但很快，他们发现坐骑装不下所有抢来的东西。 于是他们回家，召开家庭会议，寻找解决物流问题的办法。

第三天

Kwal、Pok 和 Mux 以及他们的家人，找到了物流问题的解决方案： 他们要去抢一辆马车，并把它套在他们的一匹马身上。 在他们的地区，人口稀少，资源匮乏，物资短缺。 他们生活在一片荒凉的土地上，只有从邻国流亡来的流民。 对 Kwal、Pok 和 Mux 来说，这是一片蓝海，他们梦想成为这片土地的新主宰。 问题是：他们的地盘上根本没有马车。 所以必须冒险到王国边界去偷一辆。

Kwal、Pok 和 Mux 身材高大强壮，但他们知道在王国边界， 可能会遇到经验丰富的士兵或太多农民， 这可能会改变力量对比。 于是他们决定招募附近农场的一些农民的儿子， 让他们在下次远征时陪同。 他们绑架了四个年轻人，每人来自不同的农场， 彼此并不认识，并给他们配备了简陋的装备。 这些年轻人必须步行跟随，而他们骑马。

到达王国边界后，他们发现了一个小锯木厂。 他们评估了情况：大约五个人，都是强壮的伐木工，但没有士兵。 最重要的是，他们发现了一辆装了一半木头的马车。 他们骑在马上，命令伐木工放下斧头，把马套在马车上。 伐木工很惊讶，犹豫了，他们的头领回答说如果让他们带走唯一的马车， 他们就活不过冬天，所以绝不可能。 他还试图恐吓三位巨人，说民兵很快就会来 收取送往国王城堡的木材。 没人知道他说的是真是假。

Kwal、Pok 和 Mux 毫不动摇，吹口哨召唤藏在灌木丛后的手下， 发动突袭。 Kwal、Pok 和 Mux 也加入了战斗。所有伐木工都被杀死了。 两个年轻手下战死：一个被割断颈动脉，另一个腿被砍断。 Pok 也被打伤了腿： 胫骨小骨折，但问题不大，他还能走路，只是有点跛。

Kwal 和 Mux 命令剩下的两个手下套上马车， 然后三位巨人重新上马，队伍返回老巢。 回到荒凉的家园后，Pok 回家养伤， 而 Kwal 和 Mux 放走了幸存的两个年轻农民， 并不在意去通知另外两个死者的家人。 他们现在有了马车，这才是最重要的！

第四天

Kwal、Pok 和 Mux 拉着马车去附近的农场抢粮食。 到达最近死去的年轻农民的家时，气氛沉重： 有悲伤，也有隐隐的仇恨在空气中弥漫。 Mux 注意到了这一点，并告诉了 Kwal 和 Pok。 Kwal 提议在他们的仇恨促使他们对自己或家人采取行动之前， 先把他们杀了。 Pok 并不反对，但指出不能失去太多农民， 因为他们对粮食收成至关重要，而这个地区很难吸引新居民： 人口很脆弱。 Mux 总结说，暂时最好 用恐吓手段奴役这些家庭，让他们无法反抗。 以后可以消灭他们，再用忠诚的新农民家庭取而代之。 Kwal 和 Pok 同意。他们去这些家庭，把男孩们抓走，只留下 父亲、母亲和女儿们，她们必须补偿男孩们在田里的工作。 临走前，他们说：只要有一点不轨，男孩们就会被活活撕裂。

第五天

Kwal、Pok 和 Mux 去打猎。 回来时，他们发现家人被吓坏了。 有强盗洗劫了他们的家，更可怕的是他们的女儿被强奸了。 Kwal 被血腥的愤怒所支配，但他不知道该打谁。 Pok 和 Mux 把所有家人安置在一所房子里， 然后试图安慰和安抚大家，尤其是 Kwal。 Pok 喊道：“必须找到这些强盗，但怎么找？” 沉默笼罩，大家都在思考。 Mux 有了主意：这些强盗肯定不只抢了一家， 他们一定还在附近的农场作恶。 Mux 决定去邻近的农场寻找线索。 他让 Kwal 和 Pok 留下保护家人，以防万一。 大家都很难过，但最终同意了 Mux 的提议。 Mux 去了最近的农场，三公里外：那里也被洗劫了。 鲜血、无意义的暴力、火焰、混乱：他看到被毁灭的家庭。 他再次感受到那种黑暗的情绪， 就像他看到为马车而死的年轻农民家庭时一样…… Mux 知道自己从来不是个善人，但此刻， 他明白自己有机会为被强盗摧毁的家庭带来希望。 他和他的部落可以成为当地农民的信任之源。 Mux 调查强盗留下的痕迹，试图追踪他们。 在第三个农场，他远远看到了强盗。他没有冒险，保持距离。 大约十个人，大多数骑马，显然是老练的战士。 Mux 跟踪他们一整天，直到发现了他们的营地。 现在他可以回家了。

第六天

Kwal、Pok 和 Mux 准备袭击强盗营地。 他们有两个选择：夜袭，在他们睡觉时像老鼠一样割喉， 剩下的用斧头解决，哪怕会有损失。 但他们并不擅长隐秘行动。 或者，他们可以设下埋伏，把强盗分成小队， 如果这些傻瓜真的分开，就各个击破。 Kwal 有另一个主意。 他提议招募一些农民，让强盗以为他们面对的是一个真正的部落， 并向强盗头目发起决斗。如果对方同意，Kwal 保证会打败他并收编所有人。 Pok 和 Mux 互相看了一眼，还没开口，Kwal 就咆哮道： “你们最好别想着我会输，肉疙瘩！” Pok 吐了口唾沫： “如果这些混蛋加入我们，你真的想让那些强奸了我们女人的家伙 成为部落的一员吗？” Mux 嘶哑地笑了： “他们永远不会成为部落的一员。 我们只是利用他们。 我们的王国需要战争之犬。 我们要成为荒原之王， 就需要怪物来吓唬其他怪物。”

第七天

Kwal、Pok 和 Mux 开始执行计划。 他们去邻近的农场：这次不需要强迫农民参战。 三巨人的名声和强盗造成的破坏足以让大多数村民毫不犹豫地加入他们。 几小时内，他们组建了一支约二十人的队伍，从十二岁的少年到三十五岁的男人。 装备简陋，但队伍的气势足以让人误以为是一支有组织的部队。 Kwal、Pok 和 Mux 走在队伍前列，身穿盔甲，骑马，手持长矛和斧头。

他们来到强盗营地前。 强盗们还在那里，围着篝火，享受着抢来的食物和酒。 Kwal 走向第一个守卫。守卫举剑，大喊警告营地。 Kwal 毫不犹豫，挥斧砍下强盗的手。 同时，他抓住对方的喉咙，把他提离地面，低吼道： —— 你的头是谁？ 强盗被砍断的手剧痛难忍，尽管被掐住脖子还是惨叫： —— 我的手！我的手！ 强盗营地一片混乱。 强盗们冲过来，准备保卫他们的战利品，迎战这些意外的入侵者。 Pok 和 Mux 各自带领十人，组成两个威胁性的方阵。 他们的出现足以让强盗们不敢轻举妄动。 时间仿佛静止。紧张气氛消退，被砍手的守卫脸色苍白晕了过去。 突然，一道沙哑的声音打破沉默： —— 你是谁，敢来这里捣乱！？ Kwal 把受害者扔到地上，咆哮道： —— “我是这片土地的领主！ 在这里，你没有任何权利，无论生死！ 为了避免大屠杀，我向你发起决斗。 胜者将获得这片土地。” 强盗头目并非英雄，试图拖延时间，绕开话题。 但 Kwal 不给他任何退路，逼他在手下面前承担起头领的责任。
